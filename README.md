# NEWIFI3

 自动从lean的lede源码clone并生成newifi3固件 

=============================================

本仓库每天UTC 20:00自动拉取源码编译，可直接在action中下载最新固件

固件具体更新内容移步到 https://github.com/coolsnowwolf/lede 查看

固件的默认地址为`192.168.5.1`，密码默认`password`
